# Ai_Agent_using_N8N
AI Personal Assistant Agent  An intelligent AI-powered Personal Assistant built using n8n, Groq, Streamlit, and Google Workspace APIs that automates everyday productivity tasks through natural language conversations.

-Features
 Create and manage Google Calendar events
 Read, summarize, and send Gmail messages
 Create, fetch, and delete Google Tasks
 Create and manage notes using Google Docs
 Track expenses with Google Sheets
 Perform calculations and budgeting
 Search the web for real-time information
 Conversation memory for contextual interactions
 User-friendly Streamlit chat interface
 
-Tech Stack
Python
Streamlit
n8n
Groq LLM (GPT-OSS-120B)
Google Calendar API
Gmail API
Google Tasks API
Google Docs API
Google Sheets API
Architecture

User → Streamlit Chat Interface → n8n Workflow → AI Agent → Google Workspace Tools / Web Search → Response

 Project Highlights
Built a multi-tool AI Agent capable of executing real-world actions.
Integrated multiple Google services into a single conversational assistant.
Implemented memory for context-aware interactions.
Automated workflows using n8n tool calling.
Designed a scalable architecture for future AI agent enhancements.
